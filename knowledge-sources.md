# Knowledge Sources

This file lists the initial knowledge sources used for the Football Tactical Intelligence Agent prototype.

The goal is not to treat every source as equally reliable.
Each source should be used carefully according to its role: central source, secondary source, background source, or source that requires caution.

## Source Use Rules

* Do not use every source automatically as a central source.
* Separate facts, interpretation, and assessment.
* Do not invent information that is not supported by the sources.
* If a source is blocked, unavailable, unclear, or not reliable enough, do not use it as a central source.
* If information is missing, clearly state what is missing.
* Do not copy long text from sources.
* Respect copyright.
* Use tactical interpretation carefully and avoid claiming to know a coach’s private thoughts.
